# 要件定義

## 機能要件

### 1. KIF 読み込み

- KIF ファイルを受け付ける（解析なし・水匠解析付きの両方）
- 以下を抽出する
  - 対局情報（対局日時・先後手・勝敗・手数）
  - 指し手リスト
  - 評価値・読み筋・候補手（水匠解析付きの場合のみ）
- 同一対局の重複取り込みを検出する（MVP は raw KIF の一致で判定し、将来的にハッシュや対局日時・先後手の補助判定を検討する）

### 2. SFEN 生成

- KIF 手を USI に変換する
- USI を盤面へ適用する
- KIF を再生して各手番の局面を SFEN 文字列で生成・保存する
- 初手（開始局面）から終局まで全手番分を保存する
- SFEN は戦法判定・囲い判定・定跡 DB・AI 解説の共通キーとして扱う

### 3. 評価値保存

- 手番ごとに以下を保存する（解析なし KIF の場合は全て NULL）

| フィールド   | 型      | NULL 許容 | 内容                         |
| ------------ | ------- | --------- | ---------------------------- |
| `eval`       | integer | ✅        | 先手視点の評価値（cp）       |
| `best_move`  | string  | ✅        | 水匠が示す最善手（USI 形式） |
| `pv`         | string  | ✅        | 読み筋（スペース区切り USI） |
| `candidates` | JSON    | ✅        | 候補手リスト（手＋評価値）   |

### 4. 棋譜閲覧

- 対局一覧を表示する（対局日時・先後・勝敗・手数）
- 対局を選択すると盤面ビューアを開く
- 盤面は手番を進める／戻すことができる

### 5. グラフ表示・局面連動

- 評価値がある対局は評価値の推移を折れ線グラフで表示する（先手視点）
- 評価値がない対局はグラフ非表示（手数バーのみ表示）
- グラフ上の点をクリックすると対応する手番の局面へ移動する

---

## 非機能要件

| 項目         | 目標                                      |
| ------------ | ----------------------------------------- |
| 対応ブラウザ | Chrome 最新版（個人利用のため最小限）     |
| 取り込み速度 | KIF 1 ファイル（200 手）を 1 秒以内に処理 |
| DB サイズ    | 棋譜 1,000 局・局面 200,000 件程度を想定  |
| 認証         | なし（ローカル or 個人サーバー運用）      |

---

## スコープ外（MVP）

- マルチユーザー対応
- モバイル最適化
- リアルタイム解析（水匠との直接通信）
- 棋譜の編集・コメント入力
- 外部定跡 DB の取り込み
- AI 解説生成
